myFlix Documentation

ID Request URL HTTP Method Request Response
1 Get a list of all movies /movies GET None A JSON object with complete movie list
2 Return data on movie /movies/[title] GET None Text message stating "Successful GET request returning title of movie"
3 Return genre by name /movies/genre/[genreName] GET None Text message stating "Successful GET request returning genre of movie"
4 Return director's name /movies/directors/[directors] GET None Text message stating "Successful GET request returning name of director"
5 Create new user /users POST JSON object holding new user information:
{
username : "theatergeek76",
favoriteMovies=[]
}
 Text message stating "Successful POST request adding new users"
6 Update username /users/[id] PUT JSON object holding data to update. Example:
{
"username":"dccomics099"
}
 Text message stating "Successful PUT request updating new users info"
7 Add movie to favorites list of user /users/[id]/[title] POST None Text message stating "Successful POST request updating users favorite movies"
8 Remove a movie from list of favourites /users/[id]/[title] DELETE JSON object with title of movie to be removed. Example:
{
"title": "The Avengers"
} 		Text message stating "Successful DELETE request to remove movie from users list"
9 Remove user /users/id DELETE JSON object with username of user to remove in
{
"username": "marvel123"
} 		Text message: "Successful DELETE request to remove user"
